Video Tutorials

Using the "Vamp-Off" controller is easy. Below, we have step by step videos that will guide you thru setting-up the hardware and the software.

Set up

Enter your wi-fi router SSID and password and a nickname for the Vamp-Off unit. Then start the sync procedure.

Control

Use the mobile phone app to manually turn the appliance standby power on or off.

Schedule

Use the mobile phone app to set a calendar that will automatically turn appliance standby power off or on.
